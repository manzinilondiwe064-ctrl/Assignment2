# Assignment2
This is a "Hack and Myth" app to test coomon life and coding tips
How to play
Read the statement shown on screen
Decide if it is Hack (true/helpful tip) or a Myth (false)
Tap the Hack button or Myth button to answer
You will see feedback explaning if you were right or wrong
Tap next to go to the next question 
After 5 questions, you will see your final score
Example 
Question: "Drinking water hekps you stay focused and energized"
Tap Hack because it is true
